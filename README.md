# NutriHabit – Seguimiento de Hidratación Diaria

## Descripción

NutriHabit es una aplicación móvil Android diseñada para ayudar a los usuarios a desarrollar hábitos de hidratación saludables mediante el registro del consumo diario de agua, estadísticas de progreso y recordatorios personalizados.

## Objetivo General

Desarrollar una aplicación móvil Android que permita a los usuarios registrar y monitorear su consumo diario de agua mediante recordatorios personalizados y herramientas de seguimiento que contribuyan al desarrollo de hábitos de hidratación saludables.
